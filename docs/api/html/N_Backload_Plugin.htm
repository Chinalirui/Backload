Backload.Plugin Namespace
Contains implementations for the interfaces in the Backload.Contracts.Plugin namespace. Internally used to create a client plugin specific result object.
Classes
  ClassDescription
ClientPluginExtraChunkInfo
Used in client side plugins to provide additional chunks information.
ClientPluginExtraInfo
Used in client side plugins. Provides additional information usually not provided by the client plugin by default
ClientPluginExtraInfoEx
Used in client side plugins. IClientPluginExtraChunkInfo provides folder information to the client in the "extra" field for a file in the JSON response. To get this data returned it must be enabled in the configuration (Web.Backload.config) within the ReturnExtraInfo attribute.
Remarks
Only for internal use.
 Tip
It is strongly recommended to use the inferfaces provided in the Backload.Contracts.Plugin namespace wherever possible instead of the implementations.
