  NameDescription
ChunksComplete
Returns true if all chunks for the IFileStatusItem file item have been completely uploaded.
CreateArchive(IArchiveMethodOptions)
Creates a zip archive of the files in the specified directory based on the provided IArchiveMethodOptions options and returns the result in a IArchiveMethodOperationResult result object.
CreateArchive(String, String)
Creates a zip archive of the files in the specified directory and returns the result in a IArchiveMethodOperationResult result object.
CreateArchive(IListString, String, String)
Creates a zip archive of the files in the specified directory based on the provided options and returns the result in a IArchiveMethodOperationResult result object.
CreateArchive(String, String, Boolean, Boolean, IListString)
Creates a zip archive of the files in the specified directory based on the provided options and returns the result in a IArchiveMethodOperationResult result object.
CreateDatadUrl
Creates a BASE64 encoded data URL with the byte array in the parameter and the content type.
CreateDataUrl
Creates a BASE64 encoded data URL for the provided IFileStatusItem file. FileDataSource defines if the data comes from the FileStream stream or FileData byte array.
CreateFileDataUrls
Creates BASE64 encoded data URLs for the files of the internal file status object and enforces max size settings
CreateFileDataUrls(IFileStatus, Int64)
Creates BASE64 encoded data URLs for the files in the IFileStatus parameter and enforces the maximum size of the maxSize parameter.
CreateFileResult
Returns a IFileDataResult result object containing a byte array of the file data and additional data
CreateFileResult(IFileStatus, String)
Returns a IFileDataResult result object containing a byte array of the file data for the provided IFileStatus object (first item in the file list).
CreateFileResult(IFileStatusItem, String)
Returns a IFileDataResult result object containing a byte array of the file data for the provided IFileStatusItem file.
CreateNonImageThumbnail
Creates a thumbnail for the file provided in the IFileStatusItem parameter based on the IThumbnailsConfig configuration and returns the result in a byte array.
CreatePluginResult
Creates a IFileStatusResult result object for the configured client side plugin based on the request context.
CreatePluginResult(IFileStatus, PluginType)
Creates a IFileStatusResult result object for the provided PluginType plugin type based on the IFileStatus object and request context.
CreateResult
Creates a IBackloadResult derived result object based on configuration (plugin type) and request context (e.g. GET or POST).
CreateThumbnail(IFileStatusItem)
Creates a thumbnail for the file provided in the IFileStatusItem parameter and returns the result in a byte array.
CreateThumbnail(IFileStatusItem, IThumbnailsConfig, ResizeModeType)
Creates a thumbnail for the file provided in the IFileStatusItem parameter based on the IThumbnailsConfig configuration and ResizeModeType resize mode parameters. This configuration can be retrieved from main configuration Backload.Contracts.Context.Config.
CreateThumbnailDataUrls
Creates BASE64 encoded data URLs for the thumbnails of the internal IFileStatus instance.
CreateThumbnailDataUrls(IFileStatus)
Creates BASE64 encoded data URLs for the thumbnails of the files in the IFileStatus parameter.
CreateThumbnails
Creates thumbnails for all files in the internal IFileStatus object.
CreateThumbnails(IFileStatus)
Creates thumbnails for all files contained in the IFileStatus parameter
CreateThumbnails(IFileStatus, IThumbnailsConfig, Boolean)
Creates thumbnails for all files contained in the IFileStatus parameter based on the IThumbnailsConfig configuration parameter. This configuration can be retrieved from main configuration Backload.Contracts.Context.Config.
DeleteDirectory
Deletes the directory and files physically from storage
DeleteFile(String)
Deletes the file physically from storage
DeleteFile(IFileStatusItem)
Deletes a single IFileStatusItem file.
DeleteFiles
Deletes all files of the internal IFileStatus files list.
DeleteFiles(String)
Deletes files in the specified directory and returns the result in a IDeleteMethodOperationResult result object.
DeleteFiles(IDeleteMethodOptions)
Deletes files in the specified directory based on the provided IDeleteMethodOptions options and returns the result in a IDeleteMethodOperationResult result object.
DeleteFiles(IFileStatus)
Deletes all files in the IFileStatus status parameter.
DeleteFiles(String, Boolean, Boolean, IListString)
Deletes files in the specified directory based on the provided options and returns the result in a IDeleteMethodOperationResult result object.
ExecuteDeletePipeline
Executes the complete DELETE method execution pipeline and returns a list of processed files in a IFileStatus object.
ExecuteGetPipeline
Executes the complete GET method execution pipeline and returns a list of available files in a IFileStatus object.
ExecutePostPipeline
Executes the complete POST method execution pipeline and returns a list of processed files in a IFileStatus object.
GetFile
Returns a single IFileStatusItem file object available in the current request context containing meta file information.
GetFile(String)
Returns a single IFileStatusItem file object referenced by the the provided file path containing meta file information.
GetFiles
Returns a list of available files in a IFileStatus object.
GetFiles(String, Boolean)
Returns a list of available files in a IFileStatus object.
GetFiles(String, Boolean, Boolean, IListString)
Returns a list with the absolute path of all files for the specified folder.
GetPostedFiles
Returns the meta data of the uploaded file in a IFileStatus object.
GetPostedFiles(HttpRequestBase)
Returns the meta data of the uploaded file in a IFileStatus object using the provided HttpRequestBase object.
GetPostedFiles(IBackloadDataProvider)
Returns the meta data of the uploaded file in a IFileStatus object.
GetRequestedFileItem
Returns a IFileStatusItem file object for a requested file. Only returns an object if a specific file is requested (GET or DELETE).
GetRequestedFiles
Gets the a IFileStatus instance for the requested file in the current request context. Only returns an object if a specific file is requested (GET or DELETE)
GetRequestedFiles(IBackloadContext, IRequestValuesBag)
Gets the a IFileStatus instance for the requested file in the specified request context. Only returns an object if a specific file is requested (GET or DELETE).
IsAjaxPost
Returns true if the file is posted as AJAX post. In an AJAX POST the file in not contained in a form parameter, instead the body of the request contains the file.
IsAjaxPost(HttpRequestBase)
Returns true if the file in the provided HttpRequestBase is posted in a AJAX file post.
IsFileChunk
Returns true if the current uploaded file is a file chunk.
LoadFile(String)
Loads the requested file from the physical storage into memory as a byte array. The "filepath" parameter can be a file system or web based path.
LoadFile(String, String)
Loads the requested file from the physical storage into memory as a byte array. The "filepath" parameter can be a file system or web based path. This method is usually used for content type thumbnails with an alternative web based path.
LoadFiles
Loads file data for all files available in the current execution context into the internal IFileStatus instance.
LoadFiles(IFileStatus)
Loads file data for all files available in the current execution context into the internal IFileStatus instance.
LoadFiles(IFileStatus, Boolean)
Loads file data for all files available in the current execution context into the internal IFileStatus instance. Can optionally enforce a maximum file size.
LoadFileStream
Loads file data referenced in the IFileStatusItem file from storage into the FileStream property of the provided IFileStatusItem object.
LoadThumbnails
Loads thumbnails for all files available in the current execution context into the internal IFileStatus instance.
LoadThumbnails(IFileStatus)
Loads thumbnails for all files available in the provided IFileStatus object.
MergeChunks
Merges file chunks for the IFileStatusItem file item when all chunks have been uploaded.
ParseAjaxRequest
Gets IFileStatus file meta data from a provided HttpRequestBase object (AJAX).
ParsePostRequest
Gets IFileStatus file meta data from a provided HttpRequestBase object (HTTP POST method).
ParseRequest
Gets IFileStatus file meta data from the internal data provider (e.g. HTTP request).
ParseRequest(IBackloadDataProvider)
Gets IFileStatus file meta data from a provided IBackloadDataProvider data provider object.
ProcessImage(IFileStatusItem, IImagesConfigBase, Boolean)
Processes an image in the IFileStatusItem file parameter based on the IImagesConfigBase configuration.
ProcessImage(IFileStatusItem, IImagesConfigBase, String, Boolean, Boolean)
Processes an image in the IFileStatusItem file parameter based on the IImagesConfigBase configuration and defines a content type for the resulting image
SaveBytes
Stores a byte array to the destination path. If the byte array is null, the FileData property is used as data source.
SaveChunk(IFileStatusItem)
Saves the file chunk provided in the provided IFileStatusItem object parameter. Note: This method does not merge the file chunks automatically if the chunks are complete
SaveChunk(IFileStatusItem, Boolean)
Saves the file chunk provided in the provided IFileStatusItem object parameter. Note: This method merges the file chunks if autoMerge is set to true.
SaveFile(IFileStatusItem)
Stores the file in the provided IFileStatusItem file.
SaveFile(IFileStatusItem, String)
Stores the file in the IFileStatusItem parameter to the provided path.
SaveMappingFile
Saves the mapping file for the IFileStatusItem file item. A mapping file preserves the original file name if UniqueFileNames feature is set to true in the configuration (Web.Backload.config).
SaveStream
Stores the Stream stream to the provided file path.
SaveThumbnail
Stores the thumbnail in the IFileStatusItem parameter.
