Url
Complete URL to the file handler including query strings, corresponds to HttpRequestBase .Url.
Examples
// Production: dataStore.Url = "https://uploads.mycompany/myapp/Backload/FileHandler?objectContext=..."; // Local development: dataSttore.FileHandlerUrl = "http://localhost:55555/Backload/FileHandler?objectContext=...";
UrlReferrer
Source of the request as URL
Examples
dataStore.UrlReferrer = new Uri("http://localhost:5555");
WebApplicationPath
Web application path (relative), refers to ApplicationPath. This is the name of the web application In IIS the name of the web application is concatenated to the host URL. Example: If "myapp" is the name of the web application in IIS the URL might be http://localhost/myapp/. In this case "WebApplicationPath" is "myapp/". During development IIS Express uses different ports to distinguish web applications instead of concatenating web application names to the host URL, so on IIS Express "WebApplicationPath" is usually "/".
