ICommandArgument Properties

The ICommandArgument type exposes the following members.

Properties
  NameDescription
ColumnMapping
IColumnMapping provides properties for mapping custom column names to column names used by the Backload Database plugin when executing SQL commands (usually chunked file uploads).
CommandType
CommandArgumentType provides the current command type.
Exclude
List of excluded files names or ids (Default: null)
FileContext
A IBackloadStorageProviderFile object. Used to add, update or delete the file.
FileId
A Guid file id. Used to find a file by id
InclSubFolder
Include (virtual) subfolders in a search method (Default: false)
LoadData
True to return file data {byte array) for available file(s) (Default: false)
Path
Search path (Default: "").
Pattern
A file name pattern used to filter files in a search for available files (Default: null)
SaveChanges
True to save changes immediately after the current CRUD command finishes
SqlCommand
A formatted SQL command.
SqlCommandCols
An array of column names used in the SQL command. These column names can be changed before command execution and formatted to the SqlCommandRaw SQL command string.
SqlCommandRaw
A raw SQL command with placeholders for column names.
SqlParameter
An array of SQL parameter used to execute a SQL command.These parameters can be changed before command execution and formatted to the SqlCommandRaw SQL command string.
