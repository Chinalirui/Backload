IBackloadRequestValues Properties

The IBackloadRequestValues type exposes the following members.

Properties
  NameDescription
Cache
True if the server side caching mechanism should be used.
Content
Name of a content type folder, if content type sub folders are used (e.g. movies)
FileName
File name in GET and DELETE requests
FilesFilter
A filter for files in a GET request.
Methods
List of IBackloadMethod server side methods called in a request with the "execute={method-name}" query-string parameter
ObjectContext
The current ObjectContext used to separate users or objects, usually an user id or object id.
Plugin
Name of the client side plugin explicitly set. Usually this configuration is set in Web.Backload.config.
Preview
If true, this is a request for the thumbnail.
UploadContext
One or more sub folders names (comma separated)
Uuid
An id provided as a unique file id.
