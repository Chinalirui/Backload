IFileHandler Interface
A class derived from IFileHandler represents the main type in the component and most types within the namespace are accessible in properties and sub-types. The component is fully configurable from a configuration file (Web.Backload.config) or in code (Backload.Contracts.Context.Config. The integrated execution pipeline provides a convenient way to perform all required tasks like reading the request context, process files and returning a result with only one line code. It can be executed by calling the Execute method. When running the execution pipeline, server side Events events are available for the most important execution steps during code execution. These events allow custom code to change values before the processing step executes, to change results or to log events. They also help to keep your code clean. Furthermore, high-level and low-level API's are available in the Services namespace.
