Backload.Plugin.Handler Namespace
Contains client plugin handler implementations for the interfaces in the Backload.Contracts.Plugin namespace. Internally used to create a client plugin specific result object.
Classes
  ClassDescription
FineUploader
Produces client plugin specific output for the Fine Uploader from Widen Enterprises
FineUploaderPureResultList
Internal use only. Used to return a pure list of IClientPluginFileItemCore file items for a client plugin that needs a pure list as result (e.g. Fine Uploader).
FineUploaderErrorItem
Used to return an error item for FineUploader
FineUploaderGetItem
Provides file meta data in GET requests for Fine Uploader
FineUploaderItemCore
Provides basic file meta data used in DELETE and POST requests for Fine Uploader
FineUploaderPostItem
Provides file meta data in POST requests for Fine Uploader
JQueryFileUpload
Produces client plugin specific output for the jQuery File Upload Plugin from blueimp
JQueryFileUploadItem
Stores file meta data used in POST/PUT and GET requests for the JQuery File Upload Plugin
JQueryFileUploadItemCore
Stores basic file meta data used in DELETE requests for the JQuery File Upload Plugin
PlUpload
Produces client plugin specific output for the PlUpload Plugin from MoxieCode
PlUploadItem
Stores file meta data used in POST/PUT and GET requests
PlUploadItemCore
Stores basic file meta data used in DELETE requests.
Remarks
Only for internal use.
 Tip
It is strongly recommended to use the inferfaces provided in the Backload.Contracts.Plugin namespace wherever possible instead of the implementations.
