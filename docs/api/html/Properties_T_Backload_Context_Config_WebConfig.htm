Context
If set to Global (default) the configuration settings are used by all requests. If you change a property in "global" mode, all subsequent requests use the new value. If you set this property to Request and now change a property in "request" mode, this new setting is only used in the current request.
DeleteUrlPattern
Custom URL pattern for communication with client side plugins when deleting files
Remarks
Pro/Enterprise only
EnableIntegratedHandler
True, to enable the integrated handler. If false you must provide your custom handler (see examples)
Remarks
Pro/Enterprise only
Eventing
True if events should be fired. Turn on in configuration, if you have a custom controller and want to bind event handlers to events. (Default: false)
FileExistsBehaviour
With the FileExistsType enumeration the behavior can be defined if the file already exists on uploads
FileNotFoundBehaviour
With the FileNotFoundType enumeration the behavior can be defined if a requested file could not be found
FilesUrlPattern
Custom URL pattern for communication with client side plugins when requesting files
Remarks
Pro/Enterprise only
GetInclSubFolders
True to include subfolders in the search for available files (GET requests)
GlobalConfig
If Context is set to Request, this property is a reference to the global (static) configuration as a base for all requests
Images
IImagesConfig provides access to the image configuration setting in Web.Backload.config
KeepOrgFileNames
True to store the original file name, if the UniqueFileNames feature is set,
MappingSubFolder
Sub folder name if the UniqueFileName and KeepOriginalFileName features is turned on.
Plugin
With the PluginType the name of the client side plugin can be defined. Set to "auto" if the request sends a plugin parameter.
ReturnExtraInfo
With the ReturnExtraInfoType enumeration you can define, if extra or custom data should be returned in JSON responses to the client
Security
ISecurityConfig provides access to the Security configuration setting in Web.Backload.config
Storage
IBasicStorageConfig provides the storage related configuration
Thumbnails
IThumbnailsConfig provides access to the Thumbnails configuration setting in Web.Backload.config
ThumbsUrlPattern
Custom URL pattern for communication with client side plugins when requesting thumbnails
Remarks
Pro/Enterprise only
Tracing
Enables tracing support. Note that tracing consumes some system resources and should only be enabled during development or to find errors in production. (Default: false)
UniqueFileNames
True to rename files on uploads to a unique file name ({GUID}.ext).
