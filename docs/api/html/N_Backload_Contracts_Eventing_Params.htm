Backload.Contracts.Eventing.Params Namespace
This namespace contains the definitions for the IBackloadEventParamBase and derived interfaces providing event specific parameters to registered event handlers.

Remarks
 Important
Only for internal use. The interfaces and classes in this namespace should not be used directly. At runtime you can access server side events in the Events property.
 Tip
It is strongly recommended to use the inferfaces provided in the Backload.Contracts namespace wherever possible instead of the implementations.
