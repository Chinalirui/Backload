CorsConfig Properties

The CorsConfig type exposes the following members.

Properties
  NameDescription
Credentials
Specifies if a request must send credentials by setting credentials="true" [Default="false"]
Domains
Comma-separated list of allowed domains for client side requests or "*" to allow all client side CORS requests. [Default="*"]
Enabled
True to enable CORS [Default="false"].
Headers
Comma-separated list of allowed headers. [Default="Content-Range"] (Content-Range is used for chunked file uploads in JQuery File Upload Plugin)
MaxAge
Validity period of CORS permission settings (in seconds) before the client makes a new request for permissions. Note: Not all clients use this value [Default="36000"]
Methods
Comma-separated list of allowed HTTP methods. [Default="GET,POST,DELETE"]
