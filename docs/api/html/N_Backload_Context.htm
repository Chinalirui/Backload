Backload.Context Namespace

Contains implementations of Backload's execution Backload.Contracts.Context related interfaces. The most important types in this namespace implement the IBackloadContext, which provides access to most context related types in its properties. Another type implements IWebConfig which provides access the runtime properties and settings of the static configuration (Web.Backload.Config).

Related interfaces and enumerations are available in the Backload.Contracts.Context namespace.

Classes
  ClassDescription
BackloadContext
BackloadContext enables central access to the request context and configuration environment.
BackloadMethod
Provides the name and parameters for a server side method to execute sent with a request
BackloadRequestValues
Provides Backload specific request parameters (e.g. ObjectContext or FileName)
BasePathInfo
Provides access to the root path
CorsRequest
Processes the current request for CORS support
FileInfoWrapper
Provides basic file meta data and an FileInfo object if available. A derived class can be used to create a IFileStatusItem instance.
ImageInfo
ImageInfo provides image size and resolution when the current uploaded file is an image
RequestValuesBag
Provides access to form and query values of the request. Backload specific values are available in BackloadValues
Remarks
 Important
Only for internal use. The classes in this namespace should not be used directly. At runtime you can access the Backload context in the Context property.
 Tip
It is strongly recommended to use the inferfaces provided in the Backload.Contracts.Context namespace wherever possible instead of the implementations.
