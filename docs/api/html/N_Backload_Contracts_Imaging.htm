IImageResizer Interface
Backload.Contracts.Imaging Namespace

Contains definition of the IImageResizer interface which provides image processing features like resizing, cropping, canvas, watermark or background. While the implementing class internal you can create an instance by calling the Create method on ImageResizer provided in Services.

Interfaces
  InterfaceDescription
IImageResizer
Provides methods to resize images and create thumbnails
Remarks
 Tip
It is strongly recommended to use the inferfaces provided in the Backload.Contracts namespace wherever possible instead of the implementations.
Examples
Note: In the following code example "handler" is an IFileHandler instance
Copy
var config = handler.Context.Configuration.Images;
var imageResizer = handler.Services.Factory.ImageResizer.Create(config);
// call a Resize method on the ImageResizer instance
