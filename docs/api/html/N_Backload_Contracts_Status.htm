Backload.Contracts.Status Namespace

Contains important interfaces IFileStatus which provides a list of available IFileStatusItem files processed within the current request context.

The files list can be accessed at runtime in the FileStatus property.

Classes
  ClassDescription
CustomMetaData
Used to store custom private and public meta data.
Interfaces
  InterfaceDescription
ICustomMetaData
Used to store custom private and public meta data.
IFileStatus
Provides a list of files processed in the current request
IFileStatusItem
Provides file meta data for a single file. This is the central interface to get information about a processed file in the current request
IMessageBag
Internal use only. Provides messages and codes for a IFileStatusItem object.
Remarks
 Important
Only for internal use. The classes in this namespace should not be used directly. At runtime you can access the file status in the FileStatus property.
 Tip
It is strongly recommended to use the inferfaces provided in the Backload.Contracts namespace wherever possible instead of the implementations.
