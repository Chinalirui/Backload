Backload.Services Namespace
Contains implementations of the interfaces in the Backload.Contracts.Services namespace. The classes of this namespace provide the execution pipeline, a high-level API defined in the IServiceHub interface, a low-level API defined in the ICoreServices interface and at the bottom level storage context specific services for file system, database and cloud storage only internally available
Classes
  ClassDescription
FactoryServices
This factory class enables external code or plugins to create new instances of internal classes
StorageInfo
Creates and provides storage information for a IFileStatusItem object using the current storage context and configuration settings (Web.Backload.config)
StorageInfoAccessInfos
Provides token information to access files in cloud based environments
StorageInfoChunkInfos
Provides information on file chunks for a IFileStatusItem object.
Remarks
 Important
Only for internal use. The classes in this namespace should not be used directly. At runtime you can access the services in the Services property.
 Tip
It is strongly recommended to use the inferfaces provided in the Backload.Contracts.Services namespace wherever possible instead of the implementations.
