IBackloadMappedStorageProvider Interface
Derived from IBackloadStorageProvider. Provides basic methods to be implemented by an external storage provider. This interface can be implemented by a DBContext (Entity Framework) or other classes providing a storage service. This interface can be used, when column names do not equal the default column names used by the IBackloadStorageProviderFile (custom data model). Adds a ColumnMapping property to map columns to the custom model.
