The BackloadContext type exposes the following members.

Properties
  NameDescription
BasePath
IBasePathInfo provides basic path settings
CacheManager
ICacheManager provides caching for requests.
Chunked
If true, this request is a POST request with chunked file data.
Configuration
IWebConfig provides the Backload configuration loaded from Web.Backload.config
Cors
ICorsRequest provides CORS (Cross-Origin Resource Sharing) settings for the current request
DataProvider
IBackloadDataProvider provides the current request data
Exception
Null if no exception occurred, else it provides the Exception exception.
HttpContext
Current HttpContextBase HTTP context if environment is web based.
HttpMethod
The current HttpMethod.
Plugin
Name of the client side plugin used in this request. This property can be set client and server side By default the configured value is used.
Ready
False if the context is in initialization phase else true.
Request
Current HttpRequestBase request object if environment is web based.
RequestType
RequestType provides the type of the GET requests (Default, File, Thumbnail)
Response
Current HttpResponseBase response object if environment is web based.
ResponseType
Type of JSON response: application/json, text/javascript, text/plain
StorageProvider
IBackloadStorageProvider provides basic storage methods used for example with Entity Framework
TraceManager
ITraceManager produces trace outputs when tracing is enabled.
Version
Version of the component
