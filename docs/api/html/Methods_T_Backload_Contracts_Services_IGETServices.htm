The IGETServices type exposes the following members.

Methods
  NameDescription
CreateThumbnailDataUrls
Creates BASE64 encoded data URLs for the thumbnails of the internal IFileStatus instance.
CreateThumbnailDataUrls(IFileStatus)
Creates BASE64 encoded data URLs for the thumbnails of the files in the IFileStatus parameter.
Execute
Executes the complete HTTP method execution pipeline and returns a IFileStatus result object with a list of processed file in the current request contest.
(Inherited from IHttpMethodServices.)
GetFile
Returns a single IFileStatusItem file object available in the current request context containing meta file information.
GetFile(String)
Returns a single IFileStatusItem file object referenced by the the provided file path containing meta file information.
GetFiles
Returns a list of available files in a IFileStatus object.
GetFiles(String, Boolean)
Returns a list of available files in a IFileStatus object.
LoadFiles
Loads file data for all files available in the current execution context into the returned IFileStatus instance.
LoadFiles(IFileStatus)
Loads file data for all files in the IFileStatus parameter.
LoadThumbnails
Loads thumbnails (bytes) for files available in the current execution context into the FileData property of a IFileStatusItem file.
LoadThumbnails(IFileStatus)
Loads thumbnails (bytes) for files in the IFileStatus parameter into the FileData property of a IFileStatusItem file.
