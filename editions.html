Editions

There are 6 editions available. You've most likely started already with the Standard Edition which is free of charge. The Standard Edition includes the most commonly used features like Storage locations, image processing, file bundeling, etc. The Professional, Enterprise and Global Leader Edition have all features included, while the Enterprise and Global Leader Editions ship with the source code. The licenses are not time limited and you receive updates for a 3/6/12 month period. Please note: After checkout we need some time (up to 24h). It helps us to contact us before checkout.
